El proyecto se trata de una hipotética tienda de hardware de computadora. Tendrá una barra de navegación en la parte superior mediante la cuál se podra acceder a los principales 
apartados de la página, entre los cuales tenemos:

Inicio, en donde habrán algunas imágenes de productos destacados.

Productos, en donde podremos ver y filtrar todos los productos disponibles y agregar al carrito el deseado (siendo administrador contaremos 
también con la edición y eliminación de los productos)

Registro y Inicio de sesión, para crear una cuenta y poder acceder al carrito personal para agregar productos.

Carrito, solo para usuarios logueados.

Edición de categorías, carga de productos y  gestión de usuarios, solo habilitados para usuarios con rol Administrador.

Iniciar proyecto:
Una vez cargada la base de datos, tanto en el front como en el back cargaremos los módulos con "npm install".
Luego, en el backend accedemos a la terminal y colocaremos "npm run dev" para iniciar el servidor.
Para iniciar el frontend accedemos a la terminal y colocamos "npm start".
